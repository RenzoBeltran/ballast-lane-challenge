# Pokedex Application

This is a modern Pokedex application built with React, TypeScript, and Tailwind CSS. It connects to a backend API to manage user authentication and Pokemon data.

## Prerequisites

Before starting, ensure you have the following installed:

- **Node.js**: v20.19.0 or higher (Recommended: v22.12.0+)
- **npm**: v10.x or higher

## Installation

1. Clone the repository (if applicable) or navigate to the project directory.
2. Install the dependencies:

```bash
npm install
```

> **Note**: If you encounter peer dependency warnings related to React versions or Tailwind, you can try using `npm install --legacy-peer-deps`.

## Running the Application

To start the development server:

```bash
npm run dev
```

The application will be available at `http://localhost:5173`.

## Build for Production

To create a production build:

```bash
npm run build
```

The output will be in the `dist_pokedex` (or `dist`) folder.

## linting

To run the linter:

```bash
npm run lint
```
